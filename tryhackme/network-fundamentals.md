# Network Fundamentals Lab

**Platform:** TryHackMe  
**Level:** Beginner  

## Objective
To understand basic networking concepts including IP addressing, ports, protocols, and secure network communication.

## Concepts Practiced
- TCP/IP fundamentals
- Common network ports and services
- Network security basics

## Tools Used
- Cisco Packet Tracer
- Wireshark (basic packet capture)

## Key Takeaways
- How data flows across networks
- Why misconfigured networks create security risks
- Importance of monitoring network traffic
