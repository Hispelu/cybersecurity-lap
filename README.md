# Cybersecurity Labs & Projects

I am an entry-level cybersecurity professional with hands-on training in cybersecurity fundamentals, SOC operations, vulnerability assessment, incident response, and ISO 27001-aligned security practices. This repository documents my learning journey, practical labs, and security concepts practiced through guided exercises and simulations.

- Incident response workflows
- ISO 27001 risk assessment and security policy examples
- security articles
